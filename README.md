# Contactless-Face-Mask-Recognition
In this project, efficientNetB0 is utilized for real time face mask recongition. 


### Tools 
* Python
* OpenCV
* Tensorflow
* Keras

#### Requirements 
```
- tensorflow > 1.15.2 
- keras==2.3.1
- imutils==0.5.3
- numpy==1.18.2
- opencv-python==4.2.0
- matplotlib==3.2.1
- scipy==1.4.1
```
