# Contactless-Face-Mask-Recognition
## Summary
In this project, efficientNetB0 is utilized for real time face mask recongition. 



#### Requirements 
```
- tensorflow > 1.15.2 
- keras==2.3.1
- opencv-python==4.2.0
- numpy==1.18.2
- matplotlib==3.2.1
- scipy==1.4.1
- imutils==0.5.3

```

### Tools 
* Python
* OpenCV
* Tensorflow
* Keras
* macOs BigSur Version 11.6
